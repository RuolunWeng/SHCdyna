# SHCdyna
![Screenshot of SHCdyna](screenshot/logo_SHCdyna_IFC24_v1.jpeg)

**Dynamic Compilation for Music Gesture-based SHC (Smart Hand Computer) Mobile Applications**

Developed by LiSiLoG, SHCdyna is a continuation and extension of the faust2smartphone project, first introduced at International [Faust](https://faust.grame.fr/) Conference (IFC) 2018. SHCdyna enhances the flexibility of creating musical applications through dynamic compilation, allowing musicians and developers to create, upload, and execute Faust projects in real-time on iOS devices. Building on the key concepts of customizable SHC (Smart Hand Computer) interfaces, motion control, and Faust's powerful digital signal processing (DSP) capabilities, SHCdyna opens up new possibilities for interactive music creationï¼Œworkshops in education and live performance.
